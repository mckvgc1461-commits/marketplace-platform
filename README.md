## 📦 DEPLOYMENT (CANLI YAYINA ALMA)

### 1. GitHub'a Yükle
```bash
git init
git add .
git commit -m "İlk commit"
git remote add origin https://github.com/KULLANICI_ADI/PROJE_ADI.git
git push -u origin main
```

### 2. Vercel'e Deploy Et
1. https://vercel.com → GitHub ile giriş
2. "Import Project" → Projenizi seçin
3. Environment Variables ekle (aşağıdaki gibi)
4. Deploy!

### 3. Environment Variables (Vercel'de)
```
NEXT_PUBLIC_SUPABASE_URL=https://dcnhmatuqazltqhqsfgu.supabase.co
NEXT_PUBLIC_SUPABASE_ANON_KEY=sb_publishable_VuyUnNdtMRj9IHxOW7Cq6g_wuTWQJq8

IYZICO_SECRET_KEY=sandbox-your-secret-key
NEXT_PUBLIC_SITE_URL=https://your-site.vercel.app
```

## 🎨 SUPABASE KURULUMU

1. https://supabase.com → Ücretsiz hesap
2. "New Project" → İsim ver
3. SQL Editor → `supabase-schema.sql` dosyasını çalıştır
4. Settings > API → URL ve Key'i kopyala
