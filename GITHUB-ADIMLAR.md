# 📦 GITHUB'A YÜKLEME ADIMLARI

## 1. YENİ REPOSITORY OLUŞTUR

1. GitHub'da sağ üstte **"+"** işaretine tıkla
2. **"New repository"** seç
3. Veya direkt: https://github.com/new

## 2. AYARLARI YAP

- **Repository name:** `marketplace-platform`
- **Description:** E-ticaret platformu
- **Public** seç (ücretsiz)
- **"Create repository"** tıkla

## 3. KOMUTLARI ÇALIŞTIR

GitHub sana komutlar gösterecek ama KULLANMA!

Visual Studio Code terminalinde şunu yaz:

```bash
git remote add origin https://github.com/KULLANICI-ADIN/marketplace-platform.git
git branch -M main
git push -u origin main
```

**ÖNEMLİ:** `KULLANICI-ADIN` yerine kendi GitHub kullanıcı adını yaz!

Örnek:
```bash
git remote add origin https://github.com/kerem123/marketplace-platform.git
git branch -M main
git push -u origin main
```

## 4. KULLANICI ADINI BULMAK

GitHub'da sağ üstte profil fotoğrafına tıkla → "Your profile"
URL'de göreceksin: `github.com/KULLANICI-ADIN`
